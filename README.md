# cmdctr

A modern SaaS platform starter using Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000)

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
